# Welcome to Kartal Ol

**Kartal Ol** is a registered nonprofit organization based in Poland working on artificial intelligence for education, language technology, speech recognition, and machine translation.

## What We Do

We are developing and supporting projects in:

- AI for education
- Machine translation
- Automatic speech recognition
- Language resources and datasets
- Open-source AI tools for accessibility and learning

Our goal is to make language and educational technologies more accessible, useful, and inclusive.
